盒子水平居中垂直居中怎样实现？

这种需求在我之前的项目中十分常见，随着flex兴起，，后来看blog发现这种方式也十分好

//1、定位
body{
  position:relative
}
.box{
    width:100px;
    height:100px;
  position:absolute;
  top:50%;
  left:50%;
  margin-left:-50px;
  margin-right:-50%;
}

.box{
  translateX:-50%;
  translateY:-50%;
}

//2、flex
body{
  display:flex;
  justify-content:center;
  align-item:center
}

//table本身控制文本；要求父级有固定宽高
body{
    width:500px;
    height:500px;
  display:table-ceil;
  vertical-align:center;
  text-align:center;ß
}
.box{
  display:inline-block;
}
