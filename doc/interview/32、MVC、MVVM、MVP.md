# 32、MVC、MVVM、MVP

##### MVC

- M：model代表数据模型，主要任务是操作数据。将新的数据发送到view改变视图
- V：view代表的是视图，主要任务是将数据模型转化成UI视图。传送指令到controller
- C：controller控制器，主要任务是负责处理业务逻辑。完成业务逻辑之后要求model改变状态
